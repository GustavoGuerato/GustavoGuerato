Software Enginner

About Me

Backend developer focused on systems programming and infrastructure-level software in C. I build low-level systems such as TCP servers, in-memory storage engines, and CLI tools, with emphasis on performance, memory management, and system behavior.

I design and implement systems from scratch to understand core mechanisms of networking, concurrency, and operating systems, minimizing reliance on high-level frameworks when working at the foundational level.

Current focus is modern backend engineering, including APIs, databases, and distributed systems, while maintaining a strong systems foundation.

Core Skills

Systems programming in C (memory management, data structures, pointers)

Networking (TCP/IP, sockets, client-server architectures)

Backend engineering (REST APIs, service design)

Concurrency and performance fundamentals

Projects

Key-Value Store in C over TCP (in-memory architecture)

Minimal HTTP server

CLI utilities in C

Low-level data structure implementations

Direction

Progression from single-node and in-memory systems toward distributed backend systems, with focus on scalability, consistency, and system design principles.
